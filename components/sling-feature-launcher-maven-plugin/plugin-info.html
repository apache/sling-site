Plugin Details

This report describes goals, parameters details, requirements and sample usage of this plugin.

Goals

Goals available for this plugin:

Goal Description
feature-launcher:help Display help information on feature-launcher-maven-plugin.
Call mvn feature-launcher:help -Ddetail=true -Dgoal=<goal-name> to display parameter details.
feature-launcher:start Start one or multiple Sling Feature(s).
feature-launcher:stop Stop one or multiple Sling Feature(s).

System Requirements

The following specifies the minimum requirements to run this Maven plugin:

Maven 3.8.1
JDK 11

The following specifies the minimum requirements to run this Maven plugin for historical versions:

Usage

You should specify the version in your project's plugin configuration:

<project>
  ...
  <build>
    <!-- To define the plugin version in your parent POM -->
    <pluginManagement>
      <plugins>
        <plugin>
          <groupId>org.apache.sling</groupId>
          <artifactId>feature-launcher-maven-plugin</artifactId>
          <version>1.0.2</version>
        </plugin>
        ...
      </plugins>
    </pluginManagement>
    <!-- To use the plugin goals in your POM or parent POM -->
    <plugins>
      <plugin>
        <groupId>org.apache.sling</groupId>
        <artifactId>feature-launcher-maven-plugin</artifactId>
      </plugin>
      ...
    </plugins>
  </build>
  ...
</project>

For more information, see "Guide to Configuring Plug-ins"
