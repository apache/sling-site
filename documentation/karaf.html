Running Apache Sling on Apache Karaf

NOTE: Tested and built with Apache Karaf 4.2.0.M2

Sling Karaf Features

A features repository for easy deployment of Apache Sling on Apache Karaf (see Provisioning for details).

(GitBox | GitHub)

Sling Karaf Configs

OSGi configurations for provisioning with Apache Karaf used in Sling's Karaf Features.

(GitBox | GitHub)

Sling Karaf Distribution

A distribution of Apache Sling based on Apache Karaf (Sling's Karaf Features and artifacts in a single archive).

(GitBox | GitHub)

Getting Started

You have to set serviceRequirements=disable in Karaf's etc/org.apache.karaf.features.cfg (not necessary when using Sling's Karaf Distribution).

1) Start Apache Karaf or Sling's Karaf Distribution.

2) Add the Apache Sling features repository (not necessary when using Sling's Karaf Distribution):

karaf@root()> feature:repo-add mvn:org.apache.sling/org.apache.sling.karaf-features/0.1.1-SNAPSHOT/xml/features

3) Install a Sling Quickstart feature, e.g. sling-quickstart-oak-tar or sling-quickstart-oak-mongo (requires a running and configured MongoDB):

karaf@root()> feature:install sling-quickstart-oak-tar

4) Install Launchpad Content (feature includes Composum):

karaf@root()> feature:install sling-launchpad-content

5) Browse to http://localhost:8181/.
