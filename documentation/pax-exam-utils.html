Sling Pax Exam Utilities

  • Table of Contents

NOTE: See Testing PaxExam for advanced and up-to-date test support.

Utilities to help testing Sling components with Pax Exam are available at https://github.com/apache/sling-org-apache-sling-paxexam-util

The focus is on making it easier to test Sling components against a full Sling launchpad, with minimal boilerplate code.

See the SlingRepositoryTest class for an example test.
